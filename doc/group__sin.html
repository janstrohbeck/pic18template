fp1d15_t fp1d15sin ( int16_t  i16phi)

Sinus Funktion with Lookup-Table.

Calculates the Sinus of a given angle using a Lookup-Table.
The Lookup-Table is defined only for 0 <= phi <= 90 Degrees.

Parameters
i16phiThe angle in degrees.
Returns
The sinus of the angle in 16-Bit-Integer representation. Divide by 2**15 to get the value in floating point representation.
